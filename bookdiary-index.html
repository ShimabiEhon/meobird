中島らも『頭の中がカユいんだ』徳間文庫

★★★★★

　形がないから大切にしていたものを全て盗まれてしまった――などと宣いながら中島らもは妻子を残してこっそり家出する。そして大阪の街をフーテンのように彷徨うのである。
　読んでいる最中はニヤケ笑いが止まらなかったのだけど、どう面白かったのか後から説明しようとしても言葉が出てこない。狐につままれた感じ。
　筆者は従業員が3人だけの零細企業でコピーライターをしており、CM用に狂ったネタの数々を考えてはノートに書きつける。このネタが不思議なのだ。何かこの世の真理を表しているような、それでいて何も表していないような……。
　『頭の中がカユいんだ』は中島らもの出した事実上最初の本であり、1986年に大阪書籍から出版されている。一度読んでみてほしい、と人におすすめできる作品だった。ちょっと気になったコピーネタの1つを引用して終わりにする。
”
　本屋。訪れた客。女店員。

「あの、すみません。『ハト麦茶殺人事件』っての、あります？」
「はい、ございますよ。上巻、中巻、下巻とございますが……」
「あ……。じゃ、上、中、下、全部まとめてください」
「はい、わかりました。（チンチンチンとレジの音）五千八百円になります」
「あ、じゃ、これ、六千円ね」
「はい、六千円おあずかりいたします。二百円のお返しです」
「あ、どうも……」
「…………。お客さま？」
「え？」
「犯人は……ホテルのメイドですよ」
「え⁉」
　去っていく客。ヒューッという風の音。
”

三宅英明・大角祐介『新しいLinuxの教科書』SB Creative

★★★★☆

　Linuxコマンドの初歩的な知識が(多分)網羅的に書かれた本。分かりやすいので前提知識がなくても楽しく読めた。ふと疑問に思ったことが2点あったのでメモとして書いておく。

・シェルスクリプト内でエイリアスコマンドを実行する方法
　16章に出てくるシェルスクリプト ~/bin/rootls.sh を見ていて ls -l の部分をエイリアスコマンドにできないかなと思った。

#!/bin/bash echo "root directory" cd / ls -l

　まず上の ls -l 部分を ll に書き換える。次にシェルスクリプト ~/.bash_aliases.sh を作成する。
alias ll='ls -l' 　ここで端末を以下のように叩くとエラーが出る。

$ export BASH_ENV=~/.bash_aliases $ bash ~/bin/rootls.sh alias ll='ls -l' test.sh: line 5: ll: command not found

　オプション expand_aliases をオンにしてやればいい。

$ export BASH_ENV=~/.bash_aliases $ bash -O expand_aliases ~/bin/rootls.sh

　1行めで環境変数 BASH_ENV を設定している。環境変数 BASH_ENV はbashを実行する度に呼び出されるらしい。
　2行目ではシェルオプション expand_aliases を設定している。-O expand_aliases を削除する代わりに、スクリプト内にオプション設定を追記しておいても良い：

shopt -s expand_aliases

・gitについて。mergeしたとき自動的にcommitされるのか否か
　19章で git の使い方が解説されていたのだけど、merge で conflict が起きたらファイルを直接編集してから add して commit しましょう、と書いてあるのを読んでふと( merge したら自動的に commit されるのか？)と疑問に思ったので曖昧に調べた。どうやら勝手に commit されるような、あるいは状況によっては commit されないような……？　ちゃんと調べていないので分からない。

佐藤友哉『転生！ 太宰治 転生して、すみません』星海社FICTIONS

★★☆☆☆

　玉川上水で山崎富栄と入水したはずの太宰が、現代の東京に転生。てんやわんやしつつも生きていく、そんなお話。芥川賞を寄越さなかった文壇へのルサンチマン。坂口安吾や志賀直哉らへの反発。そのような鬱屈が絶え間なく、けど気さくに表現されていた。
↓以下ネタバレ↓

　話の後半で太宰は、地下アイドル・ののたんを芥川賞作家に仕立て上げようとする。太宰が指導した後、劇的に変化したののたんのブログが面白かった。それから、作品のあとがきが良い意味でひどかった。全体を通して作者の中2病が炸裂していた。

アガサ・クリスティ著・中村妙子訳『春にして君を離れ』ハヤカワ文庫

★★★★☆

アガサが1944年にメアリ・ウェストマコット名義で発表した作品。いわゆるミステリィ作品ではない。主人公のジョーン・スカダモアは、地方弁護士の夫と健康な子供達に恵まれ、クレイミンスター地方で幸せに暮らしている。あるときバグダッドに嫁いだ娘が急病に罹ったために、ジョーンは見舞いに行った。物語はその帰路から始まる。荒天によって鉄道宿泊所に足止めを食らったジョーン。暇を持て余した彼女は、ひとりで今までの人生について回想するーー。
↓以下ネタバレ↓

　ジョーンは、自分に都合の良いことしか見たくない人である。夫のロドニーが、弁護士を辞めて農場を経営したいのだ、と言った時に、よく話を聞きもせずに一蹴した。彼女はその時のことを、良いことをした、と思っている。また、レスリー・シャーストンという女性とロドニーが、深い仲にあったことを薄々察知しながらも、そんなはずはないと目を背けてしまう。今まで、自分は相手のことを何一つ理解してこなかったのでは？　過去を思い返すごとにジョーンは疑念を膨らませていく。
　だがロドニーも奇妙である。ジョーンと離れてでも農場を経営することは選ばなかった。旅から帰ったジョーンを、ロドニーは「かわいそうなジョーン」といって迎える。ジョーンも、旅で得たはずの洞察を失ってしまう。彼らはこれからもこの"安定した関係"を続けていくだろう。４人の子供達が、一番被害を受けているように見えた。
